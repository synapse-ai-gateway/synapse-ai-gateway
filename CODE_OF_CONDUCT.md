# Code of Conduct

The Synapse AI Gateway project adopts the **Contributor Covenant v2.1** as its
Code of Conduct. The full text is maintained by the Contributor Covenant
project and is available here:

- **HTML:** https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- **Markdown source:** https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md
- **Other translations:** https://www.contributor-covenant.org/translations/

By participating in this project's community spaces — including the GitHub
repository (issues, pull requests, discussions), any chat channels, mailing
lists, and project-organised events — you agree to abide by the Contributor
Covenant v2.1.

## Scope

This Code of Conduct applies within all project spaces, and also applies when
an individual is officially representing the project in public spaces.

## Enforcement

Instances of behaviour that violates this Code of Conduct may be reported to
the project maintainers at:

> **conduct@synapse-ai-gateway.example** *(replace with your real contact before publishing)*

All reports will be reviewed and investigated promptly and fairly. Maintainers
are obligated to respect the privacy and security of the reporter.

For the consequences scale (warning → temporary ban → permanent ban) and the
full enforcement guidelines, see Section 4 of the Contributor Covenant v2.1
text linked above.
